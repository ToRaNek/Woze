	- Co2, Prix, Temps
	- Coût 
	- Symboles (kg, €, min)

Arrete:
	- départ 
	- arrivé
 	- Transport
	- Map : {Type:cout}

	- Nom (gare, aéroport, arret de bus)

Ville:
	- Nom
	- Liste : (Structure)

Plateforme:
	- Data	
	- Graph
	- Map : {Ville : Structure}
	
Utilisateur:
	- Prénom
	- Nom
	- Ville (la où il habite ou la où il est)
	- Identifiant 
	- ID
	- Liste : (Identifiants inutilisés)
